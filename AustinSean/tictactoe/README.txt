HOW TO RUN CODE:
Run the python file PlayTTT.py in the shell.
Follow the instructions that appear.

You can either play against the AI (which uses Adversarial Search),
or have the AI play against another AI. The search depth can be set
for Adversarial Search. It cannot be set for Alpha Beta Search. When
playing against the AI, a new game window will open.
